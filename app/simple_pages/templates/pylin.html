AAA testing

AAA pattern suggests that we should layout our tests in to 3 sections; Arrange, Act, Assert. Each part is responsible for a specific concern of a test case.

Arrange

In this section, we perform the setyp and initialization required for the test. This usually involves creating data required for a test case and / or setting up the mocks.

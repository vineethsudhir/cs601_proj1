Object Oriented Concepts in the calculator app

Some of the code snippets demonstrating Objected Oriented Concepts are included below

Use of Class for Encapsulation


class Summation(Calculator):
    def calculate_res(self):
        sum_values = 0.0
        for num in self.values_list:
            sum_values = Add.add(num, sum_values)
        return sum_values

Use of Static Methods


class Add:
    @staticmethod
    def add(num1, num2):
        add_res = num1 + num2
        return add_res
        return add_res

Project Structure

/.github folder contains /workflows. In this directory, there are 4 files what link the main branch to the heroku production app and other branches to heroku developement app. The heroku app name, and data such as username and password is stored in these files.

/app contains 3 folders

    /simple_pages which has html files that are being served

    /static contains scripts, images and stylesheets

    /templates contains base.html which has the navbar stored in it.

    /test contains python files that run the required tests to check the working of the app

    In the root directory there are some extra files that help setup docker, heroku, and other required files.
